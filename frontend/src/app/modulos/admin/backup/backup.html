Paso 1 — Credenciales OAuth2

Ingresa las credenciales de tu proyecto en Google Cloud Console. Son gratuitas.

Cómo obtener las credenciales (gratis)
1
Ve a console.cloud.google.com → Crea un proyecto nuevo (o usa uno existente)
2
Ve a APIs & Services → Library → Busca "Google Drive API" → Habilítala
3
Ve a APIs & Services → OAuth consent screen → Tipo: External → Completa nombre y email → Guarda
4
Ve a Credentials → Create Credentials → OAuth 2.0 Client ID
Application type: Web application
Authorized redirect URIs: agrega la URL exacta del campo "Redirect URI" de abajo
5
Copia el Client ID y Client Secret que aparecen y pégalos abajo

Termina en .apps.googleusercontent.com

Debe coincidir EXACTAMENTE con la que registraste en Google Cloud Console. Para local: http://localhost:8080/api/backup/drive/callback
